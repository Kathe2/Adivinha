# 🎯 Adivinhe o Número

Projeto desenvolvido com **HTML, CSS e JavaScript** durante o curso de **Lógica de Programação com JavaScript** da Alura.

O objetivo do projeto foi praticar conceitos fundamentais de lógica de programação e desenvolvimento web através da criação de um jogo interativo de adivinhação de números.

---

## 📌 Sobre o projeto

O jogo consiste em descobrir um número secreto gerado aleatoriamente pelo sistema. O usuário realiza tentativas e recebe dicas indicando se o número informado é maior ou menor que o valor correto até acertar.

Durante o desenvolvimento, foram aplicados conceitos essenciais de programação utilizando JavaScript puro.

---

## 🚀 Funcionalidades

✔️ Geração aleatória de número secreto

✔️ Validação das tentativas do usuário

✔️ Dicas de “maior” ou “menor”

✔️ Contador de tentativas

✔️ Reinício automático do jogo após vitória

✔️ Interface simples e interativa

✔️ Atualização dinâmica da página com JavaScript
